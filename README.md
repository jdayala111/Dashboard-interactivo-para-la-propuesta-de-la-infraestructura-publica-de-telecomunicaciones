# Dashboard-interactivo-para-la-propuesta-de-la-infraestructura-publica-de-telecomunicaciones
Dashboard interactivo para la propuesta de la infraestructura publica de telecomunicaciones

Este Dashboard es con el proposito de visualizar de mejor forma los datos.
Facilitando el trabajo de analisis y toma de decisiones sobre ellos.

La principal librería que se usa es `streamlit`, una librería de codigo abierto para facilitar la creación de aplicaciones web personalizadas.

1. Para correr la aplicación asegurate de tener Python 3.
2. Crea un entorno virtual para instalar las dependencias:
	```
	python3 -m venv .venv
	source .venv/bin/activate # En Linux/MacOS
	pip install -r requirements.txt
	```
3. Ejecuta la aplicación:
	```
	streamlit run app.py
	```
